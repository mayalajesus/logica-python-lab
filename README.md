> Laboratorio de treino mental em logica de programacao com Python.

Este repositorio foi criado para praticar raciocinio com problemas progressivos, do aquecimento ate desafios no estilo LeetCode, Harvard/CS50 e listas classicas de algoritmos.

Sem gabarito. Sem exemplos prontos. Apenas problemas para pensar, quebrar a cabeca e evoluir.

## Mapa do Lab

| Nivel | Tema | Foco |
| --- | --- | --- |
| 1 | Aquecimento | condicionais simples |
| 2 | Condicionais compostas | decisoes com multiplas regras |
| 3 | Loops basicos | repeticao, acumuladores e contadores |
| 4 | Loops com decisao | validacao, sentinelas e calculos |
| 5 | Strings com loops | texto, caracteres e padroes |
| 6 | Listas | sequencias, busca e comparacao |
| 7 | Loops aninhados | matrizes e formas |
| 8 | Desafios integrados | problemas maiores |
| 9 | Estilo LeetCode | funcoes e eficiencia |
| 10 | Desafios longos | mini-sistemas e simulacoes |

## Arquivos

- [`01_condicionais_e_loops.ipynb`](01_condicionais_e_loops.ipynb): notebook principal com 80 atividades.
- [`.github/workflows/notebook-check.yml`](.github/workflows/notebook-check.yml): validacao automatica do notebook no GitHub Actions.

## Como Usar

1. Abra o notebook no GitHub, Jupyter, VS Code ou Google Colab.
2. Resolva uma atividade por vez.
3. Escreva sua resposta na celula abaixo do enunciado.
4. Crie seus proprios testes.
5. Faça commits conforme avancar.

- Resolva sem olhar gabaritos.
- Priorize condicionais e loops.
- Evite funcoes prontas quando o objetivo for treinar a logica manual.
- Refatore depois de fazer funcionar.
- Documente seu raciocinio apenas quando isso ajudar a revisar depois.
